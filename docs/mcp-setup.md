# MCP — Model Context Protocol

Los servidores MCP dan herramientas extra al asistente de IA (navegador, bases de datos,
etc.). Se registran en `.mcp.json` (scope de proyecto).

## Configuración

El archivo `.mcp.json` en la raíz del proyecto define los servidores MCP disponibles.
Ver `.mcp.json.example` para ejemplos.

## Servidores recomendados

- **Playwright** — navegador headless para verificar UI
- **Supabase** — acceso directo a la base de datos

## Verificar

Revisa la configuración MCP desde el panel de Kiro o el archivo `.mcp.json`.
